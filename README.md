# GPT-2 AI Story Generator

This project uses the GPT-2 transformer model to generate creative AI-based stories from text prompts.

## Overview
GPT-2 is a pre-trained language model developed by OpenAI.  
The model predicts the next words in a sentence and generates human-like text based on the given input prompt.

In this project, GPT-2 is used to generate fantasy and science-fiction stories automatically.

---

## Technologies Used
- Python
- Hugging Face Transformers
- PyTorch
- Google Colab

---

## Features
- Generates AI-based stories
- Uses pre-trained GPT-2 model
- Simple and beginner-friendly implementation
- Supports custom prompts
